html
br - 줄바꿈 태그. 코드 내에서 enter로 줄바꿈 한다고 바뀌지 않기에 br으로 줄바꿈 해야함
hr - 줄바꿈 및 수평으로 된 줄을 긋는 태그
link, meta - head 태그 안에 들어가는 요소
<tag /> - 안전하기 때문에 많이 쓰임.
.html 확장자 처음 생성시 ! + enter 하면 기본 양식 생성됨
제목 태그 h1~6 - 웹 검색 엔진이 제일 먼저 검색하는 태그
ul ol - 목록 태그 ul은 순서 x / ol 은 순서 o & 속성 사용 가능
ul type - disc(default) / none / square
ol type - 1(default) / a / A / i / I
주석 단축키 ctrl + /(해제도 동일)
&nbsp - 띄어쓰기 엔티티 코드(많은 공백이 필요할 때 주로 사용)
img tag - src : 이미지의 위치나 주소를 지정 / alt - 이미지를 불러오지 못했을 경우 해당 이미지가 무엇인지에 대한 설명을 적을 수 있음
img 경로 - ./~~~ : 현재 위치로부터 경로를 찾아나가는 것(상대경로) / /... : 절대 경로
a tag - href: 필수 속성, 이동할 페이지의 링크 / target 새 창으로 열 것인지 새 탭으로 열 것인지
buttom - submit / button 2개 타입 -> submit : 누를시 페이지를 새로 고침 / button : 새로고침 x
label - 이 안에 글자와 input 태그를 같이 써주는 것이 호환성에 좋음 / for 속성 사용시 input의 id 와 맞춰주어야 함께 동작
